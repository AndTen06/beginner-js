## Instructions

1. Install dependencies:
   ```
   npm install
   ```

2. Run tests for all exercises:
   ```
   npm test
   ```

3. Run tests by difficulty level:
   ```
   npm run test:easy
   npm run test:medium
   npm run test:hard
   ```

4. Run tests for a specific file:
   ```
   npx jest tests/easy/ex1.test.js
   ```
